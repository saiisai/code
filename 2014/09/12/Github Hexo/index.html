Windows7搭建Blog简易教程 必须在Github注册账号以及创建repository
by GDZ

教程如下

安装 Github Node.js

下载 Github Node.js 然后安装即可

Download Link: Github Node.js

安装 Hexo (在Git Bash进行)

  1. npm install -g hexo
  2. hexo init (在放Hexo的目录下 可自建)
  3. npm install
  4. hexo generate
  5. hexo server

浏览器输入localhost:4000查看

设置ssh keys

输入以下字段(Git Bash) 然后回车

$ ssh-keygen -t rsa -C “邮件地址@youremail.com”

输入账号密码(密码无显示) 见到有类似图像则成功

Enter passphrase (empty for no passphrase):<输入加密串>
Enter same passphrase again:<再次输入加密串>

打开本地
C:\Documents and Settings\Administrator.ssh\id_rsa.pub (win7:C:\Users\Administrator.ssh)

复制id_rsa代码到github
Account Settings—>SSH Public keys —> add another public keys

测试及设置

  1. $ ssh -T git@github.com
  2. The authenticity of host 'github.com (207.97.227.239)' can't be established. RSA key fingerprint is 16:27:ac:a5:76:28:2d:36:63:1b:56:4d:eb:df:a6:48. Are you sure you want to continue connecting (yes/no)? yes
  3. Hi cnfeat! You've successfully authenticated, but GitHub does not provide shell access.
  4. $ git config —global user.name “username”//用户名
  5. $ git config —global user.email “username@gmail.com”//填写自己的邮箱

最后

Hexo目录下编辑_config.yml 最后加入以下字段

deploy:
type: github
repository: https://github.com/username/username.github.io.git
branch: master

再次执行命令

  1. hexo generate
  2. hexo deploy

浏览器访问username.github.io

主题替换

各种主题都有 替换方法link内有
Link: Themes
个人使用主题

注意

hexo现在支持更加简单的命令格式了

  • hexo g == hexo generate
  • hexo d == hexo deploy
  • hexo s == hexo server
  • hexo n == hexo new
  1. deploy的空格请留意 最好用vim来观看 不然产生错误
  2. Github里的op设置有Git Bash
  3. username与email需要一致
  4. 文章发表有问题的可以删除目录下的db.json重新开server
